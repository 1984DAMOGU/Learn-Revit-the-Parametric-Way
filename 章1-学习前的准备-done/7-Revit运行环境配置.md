# 计算机的配置 #

普通的台式机（4G内存），和普通的笔记本电脑就可以了。当然，如果你为了追求更快的速度，CPU越快，内存越多，并且能使用固态硬盘就更好了。

# 操作系统： #

Windows 7 或 8 或更高版本的64位操作系统。

# Revit版本： #

安装Revit的2014或更新的版本（我的示例视频都是用2014版本录制的）
